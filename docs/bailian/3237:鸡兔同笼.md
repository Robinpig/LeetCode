# 3237:鸡兔同笼

## Description

一个笼子里面关了鸡和兔子（鸡有2只脚，兔子有4只脚，没有例外）。
已经知道了笼子里面脚的总数a，问笼子里面至少有多少只动物，至多有多少只动物

输入
第1行是测试数据的组数n，后面跟着n行输入。每组测试数据占1行，每行一个正整数a (a < 32768)

输出
输出包含n行，每行对应一个输入,包含两个正整数，第一个是最少的动物数，第二个是最多的动物数，两个正整数用一个空格分开
如果没有满足要求的答案，则输出两个0。

样例输入
2
3
20

样例输出
0 0
5 10



## Solutions

通过脚来判断动物数量 因为鸡和兔子的脚都是偶数 所以为奇数则报错

数量最小则先取余4 , 最大数始终 除2

```java
import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner in = new Scanner(System.in);
        int length = in.nextInt();
        for (int i = 0; i < length; i++) {
            int feets = in.nextInt();
            if ((feets & 1) == 1) {
                System.out.println("0 0");
            } else {
                System.out.println(feets / 4 + (feets % 4 == 0 ? 0 : 1) + " " + feets / 2);
            }
        }
    }
}
```
